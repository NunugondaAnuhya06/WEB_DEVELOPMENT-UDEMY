CSS SELECTORS

Applying CSS to Different Parts of HTML

1.ELEMENT SELECTOR - The ELEMENT SELECTOR targets elements based on their HTML tag name.

  1. CLASS SELECTOR - The CLASS SELECTOR target elements based on the value of the class attribute.
  2. ID SELECTOR - The ID SELECTOR target the elements based on the value of the id attribute.
  3. ATTRIBUTE SELECTOR - The ATTRIBUTE SELECTOR targets the elements based on their attributes and values.
  4. UNIVERSAL SELECTOR - The UNIVERSAL SELECTOR targets all elements.
